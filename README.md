# AI Resume Analyzer 🤖

An AI-powered resume analyzer that scores resumes,
identifies strengths and weaknesses, and recommends
job roles based on role alignment.

## Tech Stack
- Frontend: React.js + Vite + Tailwind CSS
- Backend: Node.js + Express.js
- AI: Groq LLM + Prompt Engineering
- Dev Tool: Cursor AI

## Features
- ✅ Resume scoring system
- ✅ Strengths and weaknesses analysis
- ✅ Job role recommendations
- ✅ PDF resume upload support
- ✅ Clean and modern UI

## How to Run Locally
### Backend
cd backend
npm install
cp .env.example .env
# Add your GROQ_API_KEY in .env
npm start

### Frontend
cd frontend
npm install
npm run dev
